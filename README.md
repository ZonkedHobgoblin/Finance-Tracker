# Finance-Tracker
Android 16 based application to assist in tracking finance via a weekly allowance system, designed around tracking payments from mobile wallets such as Google Pay.
A notification is constantly displayed, displaying the remaining allowance for the week (Set by the user)
If a payment/wallet app (Conifgurable list of apps) sends a notifcation of a payment, the total listed in the notification is automatically deducted within the notification and app.
The app contains a history of tracked notifications.
